## Description

Access your Notion snippets instantly with hybrid search. Sync code snippets, canned responses, and bookmarks from multiple Notion databases. Features 0-latency local caching for recent items and global cloud search for your entire history. Seamlessly import snippets into Raycast's native Snippet Manager.
